Minimum and Maximum Finder:
A Python program that reads numbers from a file and finds the minimum and maximum values. The program includes robust error handling for various edge cases and invalid inputs.

Features
File Reading: Reads numbers from a text file (one number per line)
Error Handling: Comprehensive error handling for:
File not found
Empty files
Invalid/non-numeric values
Files with no valid numbers
Data Validation: Automatically skips invalid entries while providing feedback

Clear Output: Well-formatted results showing the processed numbers and calculated values
